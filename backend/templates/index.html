Vehicle Parking System API

Backend API server is running successfully!

Available Endpoints:

POST /api/auth/login - User login
POST /api/auth/register - User registration
GET /api/parking-lots - Get all parking lots
POST /api/parking-lots - Create parking lot
GET /api/parking-lots/{id}/spots - Get parking spots
POST /api/bookings - Create booking
GET /api/users/{id}/bookings - Get user bookings
GET /api/analytics - Get analytics data

Frontend: Please run the Vue.js frontend from the frontend folder to access the full application.
